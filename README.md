# FDF #

## In this project you can see 2D image from map ##
(map - it's some file with data about 'x', 'y' and 'z' coordinates).

## Used with mlx version: X11 ##
# Usage:  #
* $> make
* $> ./fdf maps/filename.fdf

## *Attention:* You must install 'minilibx' on your device ##
